# QuizZapp

QuizZapp - это простой квиз на JavaScript, который позволяет пользователям проверять свои знания в различных областях. Квиз содержит несколько вопросов с вариантами ответов, и в конце отображает результаты.

## Функции

- Интерактивный интерфейс с вопросами и вариантами ответов.
- Подсветка выбранных и правильных ответов.
- Возможность начать квиз заново.
- Отображение результатов в конце квиза.
